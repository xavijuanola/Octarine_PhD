# Octarine - Personal and PhD Notes

**Octarine** is a note-taking application for macOS that allows you to store and sync your personal and PhD notes in a Git repository. This repository is designed to keep your notes organized and version-controlled, ensuring that you can easily access, track, and modify your notes over time.

## Purpose

This repository will serve as the central hub for storing my personal and academic notes related to my PhD research. Notes are versioned and backed up via Git, providing a reliable way to keep track of changes, progress, and ideas.

## Features

- **Git Version Control:** All notes are stored and tracked using Git, allowing you to see changes over time.
- **Octarine Integration:** Octarine is used for capturing, editing, and organizing notes.
- **Markdown Support:** Notes are stored in markdown format, making them easily readable and editable.
- **Automatic Syncing:** Notes will be synced to this repository automatically, ensuring everything is backed up.

## How to Use

### 1. Clone the Repository

To get started, clone this repository to your local machine:

```bash
git clone https://github.com/yourusername/octarine-notes.git
cd octarine-notes
```
